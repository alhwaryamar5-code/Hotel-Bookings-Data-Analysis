# 🏨 Hotel Bookings Data Analysis (Python)

## 📌 Project Overview
This project performs an End-to-End Data Analysis on a hotel bookings dataset using **Python**. The goal is to analyze booking behaviors, cancellation rates, and pricing patterns to provide actionable insights for hotel management.

## 🛠️ Tech Stack & Libraries Used
* **Pandas & NumPy:** For Data Cleaning and handling missing values.
* **Matplotlib & Seaborn:** For Exploratory Data Analysis (EDA) and visualizations.

## 📊 Key Insights & Findings
1. **High Cancellation Rate:** The overall cancellation rate is **27.5%**. City Hotels face a higher cancellation rate (**30%**) compared to Resort Hotels (**23.5%**).
2. **Seasonality:** **July and August** are the absolute peak months for hotel demand.
3. **Pricing (ADR):** **City Hotels** command a higher Average Daily Rate (**~$111**) compared to Resort Hotels (**~$99**).
